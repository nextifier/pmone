Projects are the top-level containers in PM One. Every event, brand assignment, and piece of content lives inside a project. You will typically create one project per exhibition brand (e.g., "Megabuild Indonesia").

## What you will learn

- How to create a new project
- How to edit and reorder projects
- How to delete and restore projects

## Creating a project

1. Go to **Projects** in the sidebar
2. Click **Create** in the top-right corner
3. Fill in the project name, slug (URL-friendly identifier), and description
4. Upload a project logo if you have one
5. Click **Save**

The slug becomes part of your project URL and your event website's API path. Pick something short and readable, like `megabuild` or `indobuildtech`.

## Editing a project

Click any project from the list to open it. From the project page, go to **Settings** to change the name, slug, description, or logo.

## Reordering projects

Drag and drop projects in the list to change their display order. The order persists across sessions.

## Deleting and restoring

To delete a project, click the three-dot menu on the project card and select **Delete**. Deleted projects move to the trash.

To restore: go to **Projects > Trash**, find the project, and click **Restore**. You can also permanently delete from the trash if you are sure you no longer need it.

## Importing and exporting

Use the **Import** button to upload projects from an Excel template. Download the template first to see the expected format.

Use **Export** to get a spreadsheet of all your projects.

## Common questions

**Q: What happens to events inside a deleted project?**

They move to the trash with the project. Restoring the project restores its events too.

**Q: Can I transfer a project to another team?**

Not directly. You would need to adjust project membership manually.
