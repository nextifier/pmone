Different exhibitions need different data from exhibitors. Custom fields let you define project-specific questions, while business categories help you classify brands by industry.

## What you will learn

- How to create and manage custom fields
- How to set up business categories
- How these show up on brand profiles

## Custom fields

### Creating a custom field

1. Open your project and go to **Settings > Brand fields**
2. Click **Add field**
3. Set the field label, type (text, select, checkbox, etc.), and whether it is required
4. Click **Save**

Custom fields appear on the brand-event profile form. When exhibitors or staff edit a brand's event-specific info, these fields show up alongside the standard ones.

### Reordering fields

Drag and drop fields in the list to change their display order on the form.

### Editing and deleting

Click a field to edit its label or options. Click the delete icon to remove it. Deleting a field removes its data from all brand profiles in that project.

## Business categories

Business categories let you tag brands by industry (e.g., "Building Materials", "Interior Design", "Construction Equipment").

### Managing categories

1. Go to your project and find **Business categories** in the sidebar
2. Click **Add category** to create a new one
3. Drag and drop to reorder them

### Import and export

You can import categories from an Excel file using the **Import** button, or export your current list with **Export**. Download the template first if you are importing.

## How they connect

When you add a brand to an event, staff can tag it with business categories and fill in custom fields. These details appear on the event website's exhibitor listing and help visitors filter brands by category.

## Common questions

**Q: Can I share custom fields across projects?**

No. Custom fields are project-specific. If you need the same field in multiple projects, you will have to create it in each one.

**Q: What happens to category data if I rename a category?**

Brands tagged with the old name update automatically. Renaming does not break existing assignments.
