确认了需要完成的文件后，接下来就是逐一提交。有些需要勾选同意，有些需要上传文件。

## 本文内容

- 如何同意活动规则
- 如何上传文件
- 如何重新提交

## 同意活动规则

1. 打开活动规则文件
2. 阅读全部内容
3. 勾选底部的同意复选框
4. 点击 **Submit**

系统会记录您的姓名、日期和设备信息作为同意凭证。

## 上传文件

1. 打开运营文件
2. 阅读说明，了解具体要求
3. 点击 **Upload** 选择文件
4. 如有文字填写项，一并填写
5. 点击 **Submit**

## 重新提交

如果需要更新提交内容（上传了错误文件、需要更新许可证等）：

1. 重新打开该文件
2. 上传新文件
3. 点击 **Submit**

新提交会替换之前的内容。

## 常见问题

**问：支持哪些文件格式？**

通常支持 PDF、JPG 和 PNG。请查看文件说明中的具体要求。

**问：已同意的文件又变成待处理了。**

主办方更新了文件内容，需要重新查看并同意。

**问：同事可以代替提交吗？**

可以。品牌的任何团队成员都可以提交文件。
