When you have dozens or hundreds of exhibitors, manual entry is not practical. The import/export feature lets you work with brand data in spreadsheets.

## What you will learn

- How to import brands from Excel
- How to export brand data
- How to handle import errors

## Importing brands

### Global brand import

1. Go to **Brands** in the sidebar
2. Click **Import**
3. Download the template file first
4. Fill in brand data following the template's column structure
5. Upload the completed file

The importer creates new brands for names it does not recognize and updates existing ones that match.

### Event-specific brand import

You can also import brand-event assignments (brands with booth details) from within an event:

1. Open the event's **Brands** tab
2. Click **Import**
3. The template here includes event-specific columns like booth number and type
4. Upload and the system assigns brands to the event with the specified booth details

## Exporting brands

Click **Export** from either the global brand list or an event's brand list. You get an Excel file with all brand data, which you can filter, edit, and re-import.

## Handling errors

If the import encounters issues (missing required fields, duplicate entries), it reports errors with row numbers. Fix the flagged rows in your spreadsheet and re-upload.

## Common questions

**Q: What file format does the import accept?**

Excel (.xlsx) files. Use the template to get the right column headers.

**Q: Can I import brand team members along with brand data?**

The brand import handles company data. Team member assignments are managed separately through the brand profile or invitation system.
