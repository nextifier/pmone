Event documents are the rules, agreements, and forms exhibitors need to review and sign off on. You create them as staff, and exhibitors acknowledge or submit files through their dashboard.

## What you will learn

- The difference between event rules and operational documents
- How to create a document
- How to track submissions

## Types of documents

### Event rules

These are checkbox-style agreements. Exhibitors read the content and check a box to confirm they agree. The system tracks who agreed, when, and from which device.

If you update the content of an event rule, all previous agreements are invalidated and exhibitors must re-agree.

### Operational documents

These require exhibitors to upload files (permits, insurance certificates, floor plans) or fill in text responses. The system tracks each submission with the submitter's name and timestamp.

## Creating a document

1. Open your event and go to **Operational > Documents** (or through the event content area)
2. Click **Create**
3. Fill in:
   - **Title** - what this document is about
   - **Content** - the full text of the rule or instructions
   - **Type** - event rule (checkbox) or operational document (file upload/text)
   - **Applies to** - which booth types this document applies to (leave blank for all)
4. Click **Save**

## Reordering documents

Drag and drop to control the order exhibitors see documents in their dashboard.

## Tracking submissions

From the document list, you can see how many exhibitors have completed each document. Click a document to see individual submission details: who submitted, when, and what files they uploaded.

## Version tracking

When you update the content of an event rule, the system detects the change and marks all previous agreements as outdated. Exhibitors see a notification to re-agree.

## Common questions

**Q: Can I make a document required for some booth types but not others?**

Yes. Use the "Applies to" field to select specific booth types. Exhibitors with other booth types will not see the document.

**Q: What if an exhibitor uploads the wrong file?**

They can re-submit from their dashboard. The new submission replaces the old one.
