The static QR code generator creates QR codes from any text or URL you enter. Unlike the dynamic QR codes attached to short links, these encode the content directly and cannot be changed after printing.

## What you will learn

- How to generate a static QR code
- When to use static vs. dynamic QR codes

## Generating a QR code

1. Go to **QR Code** in the sidebar (under Tools)
2. Enter the URL or text you want to encode
3. The QR code generates instantly as you type
4. Click **Download** to save it as an image

## Static vs. dynamic

**Static QR codes** encode the content directly. If you print one and later want to change the destination, you need to reprint.

**Dynamic QR codes** (from short links) point to a redirect URL. You can change the destination without reprinting the QR code.

Use static for permanent content (Wi-Fi passwords, fixed URLs). Use dynamic for campaigns where the destination might change.

## Common questions

**Q: What formats can I download the QR code in?**

PNG image format. The generated image is high-resolution and suitable for print.

**Q: Is there a character limit?**

QR codes can hold up to a few thousand characters, but shorter content produces cleaner, easier-to-scan codes. Keep URLs short when possible.
