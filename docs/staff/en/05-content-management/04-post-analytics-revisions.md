PM One tracks views for each post and keeps a revision history so you can see what changed and when.

## What you will learn

- How to view post analytics
- How to use the revision history
- How to compare versions

## Post analytics

### Per-post analytics

Open a post and click the **Analytics** tab to see:

- View count over time
- Traffic sources (if tracked)

### Overall analytics

Go to **Posts > Analytics** to see aggregate numbers across all your posts: total views, top-performing posts, and trends.

## Revision history

Every time you save a post, PM One creates a revision. You can browse previous versions and compare them.

### Viewing revisions

1. Open a post in the editor
2. Look for the **Revisions** section
3. Click any revision to see what the post looked like at that point

### Comparing versions

Select two revisions to see a side-by-side diff. This is useful when multiple people edit the same post and you want to track what changed.

## Autosave revisions

Autosaved drafts are stored separately from manual saves. You can view and restore autosaved versions if you need to recover work that was not manually saved.

### Discarding autosaves

If the autosaved version is outdated or unwanted, you can discard it to clean up the revision list.

## Common questions

**Q: Can I restore an older version of a post?**

Yes. Open the revision you want and click restore. The current version becomes a new revision so nothing is lost.

**Q: How long are revisions kept?**

Revisions are kept indefinitely. There is no automatic cleanup.
