Once you know which documents are required, it is time to complete them. Some need a checkbox agreement, others need file uploads.

## What you will learn

- How to agree to event rules
- How to upload document files
- How to re-submit if needed

## Agreeing to event rules

1. Open the event rule document
2. Read the full content
3. Check the agreement checkbox at the bottom
4. Click **Submit**

The system records your name, the date, and your device information as proof of agreement.

## Uploading documents

1. Open the operational document
2. Read the instructions to understand what is required
3. Click **Upload** and select your file
4. Add any text responses if the form includes text fields
5. Click **Submit**

## Re-submitting

If you need to update a submission (uploaded the wrong file, need to update a permit):

1. Open the document again
2. Upload the new file
3. Click **Submit**

The new submission replaces the previous one.

## Common questions

**Q: What file formats are accepted?**

PDF, JPG, and PNG are generally supported. Check the document instructions for specific requirements.

**Q: I agreed but the document shows as pending again.**

The organizer updated the document content. You need to review the changes and agree again.

**Q: Can my colleague submit documents on behalf of our brand?**

Yes. Any team member linked to the brand can submit documents.
